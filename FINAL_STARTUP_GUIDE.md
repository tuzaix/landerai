## 🔧 技术栈
- **前端**: Next.js 14 + React 18 + TypeScript + Tailwind CSS
- **后端**: Python + FastAPI + Pydantic + SQLAlchemy
- **数据库**: MySQL + Redis
- **部署**: Docker容器化
